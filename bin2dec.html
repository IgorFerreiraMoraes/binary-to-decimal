What is Binary Code?

Binary code is a code used in digital computers to represent text, computer processor instructions, or any other data. It is based on a binary number system in which there are only two possible states, off and on, usually symbolized by 0 and 1. Whereas in a decimal system, which employs 10 digits, each digit position represents a power of 10 (100, 1,000, etc.), in a binary system each digit position represents a power of 2 (4, 8, 16, etc.).

Even the binary code only representing numbers, we have ways to store letters. Each letter has its number, for example, A is 65 (01000001) and Z is 90 (01011010) and space is 0 (00000000)

How to use this site?

Here we have a byte (unit of digital information that most commonly consists of eight bits), each of the lamps is a bit. Considering the ON lamps are 1s and the OFF ones are 0s, we can go from 00000000 to 11111111, or from 0 to 255, by turning them on or off. The result will be the sum of the turned on bits.
